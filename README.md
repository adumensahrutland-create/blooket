## 🚀 How to use the software

Open the application by clicking the desktop icon. The main window shows several options for your game.

### Logging in
Enter your Blooket username and password in the provided fields. The application transmits this data securely to the Blooket servers to verify your account. It does not store your password on your local hard drive.

### Choosing a game utility
You will see a list of available tools. These include game flooders, point multipliers, and cosmetic unlockers. Click the button next to the tool you wish to activate. The interface will show a confirmation message once the tool is active.

### Managing game sessions
You can track your current game session through the status window. If you wish to stop a utility, click the Stop button located in the bottom corner of the dashboard. The application resets your settings to the default state instantly.

## 📋 Troubleshooting common issues

If the application fails to launch, verify that you installed the latest version. Sometimes, antivirus software blocks new applications. Check your antivirus settings to allow the blooket.exe file to run.

If the tools do not seem to work, log out and log back in. This refreshes your session token. Ensure your Blooket account is not currently locked or restricted due to previous activity.

If the interface freezes, use your task manager to end the process. Right-click your taskbar, select Task Manager, find the blooket icon, and click End Task. You can then restart the application normally.
